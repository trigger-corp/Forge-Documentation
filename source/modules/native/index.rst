.. _native_plugins:

Native plugins
==============

Native plugins allow you to write native Android and iOS code, compile it and include this compiled code in your app built using Forge. It also provides a method for your native code to communicate with your Javascript code.

.. important:: Plugins are currently a beta feature, get in touch with support@trigger.io if you're interested in trying them out.
